Distributed Implementation of Seizure Detection Algorithm
#DistributedML #RandomForest #SignalProcessing #Spark

In this study, we explored the cross-disciplinary application of distributed machine learning in healthcare sensing. The goal was to develop a distributed implementation for seizure detections algorithms based on canine and human electroencephalogram (EEG) recordings provided by Mayo Clinic. The algorithm was designed to determine if a given EEG signal clip was within seizure stage, early seizure stage (within 15s before seizure), or neither. Driven by the need to incorporate large-scale data labeling and potential application in real-time detection, this independent study further extended the algorithm to big data scenarios by building distributed training and prediction pipelines. One potential application of this study is a data annotation system capable of annotating massive unlabelled EEG data in parallel. Another real-world end product example could be a wearable low-latency seizure prevention device that processes real-time EEG signals as streaming data.

Moving Object Recognition and Tracking in Videos
#ComputerVision #R-CNN #OpticalFlow #ObjectRecognition #OpenCV

Object recognition is refers to a collection of related tasks for identifying objects in digital photographs. In this course project, we devised approaches to recognize and track objects in videos without having to exhaustively feed each frame to object detection NN model to reduce computation costs. We parsed videos with OpenCV and employed a pre-trained Mask R-CNN model to recognize objects from 81 possible classes and generate bounding boxes in first frames of videos. Then we coded up an iterative image processing procedure to track objects in subsequent frames with KLT optical flow method based on inherited bounding boxes from first frame and identified moving objects in sliding cameras.
